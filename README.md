# Visya

Plataforma de análise avançada de dados para visão estratégica do negócio.

Transformando informação em decisões seguras e resultados consistentes.

## Stack Técnico

- **Frontend**: HTML/CSS/JavaScript (React)
- **Backend**: Node.js
- **Banco de Dados**: SQL Server
- **Cloud**: Azure/Vercel

## Instalação

```bash
git clone https://github.com/y22pdf86w4-crypto/visya.git
cd visya
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Deploy

Conectado ao Vercel para CI/CD automático.
